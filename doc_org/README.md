# タスクログ記録アプリ (TaskLogger)

デスクトップで素早くタスクの実績を記録し、後で生成AIに要約させるためのログ用Markdownを生成する、シンプルな「Fire & Forget」型アプリケーションです。

## 特徴
- **爆速のUX**: タスクを入力して `Ctrl+Enter` を押すだけで即座に保存・アプリ終了。
- **Teamsのようなチャット風入力**: 過去のタスクを参照しながら素早く追記構成。
- **自動サジェスト機能**: 過去に入力したタグをキーボード操作のみで即座に呼び出し可能。（表現の揺れを防止）
- **Markdown出力**: 月ごとに `YYYY-MM_log.md` 形式で保存されるため、そのままChatGPTやClaudeなどのAIに読み込ませて期末レポートの要約に利用しやすくなっています。

## 開発環境と依存関係
- Python 3.x
- [CustomTkinter](https://customtkinter.tomschimansky.com/) (モダンなUI用フレームワーク)

## セットアップ手順

1. リポジトリをクローンまたはダウンロードします。
2. 依存ライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```

## 起動方法
以下のコマンドでアプリを起動します。（Windows環境ではタスクバーなどにピン留めしておき、必要なときだけ起動する運用を推奨します）
```bash
python main.py
```

## 使い方
1. 表示された画面下部の入力欄で、まず「タグ」を入力します。文字を打つと過去のタグがサジェストされるので、`↑` `↓` キーと `Enter` で選択できます。
2. その下のタスク内容欄に完了した作業やメモを記述します。
3. `Ctrl+Enter` を押す（またはボタンをクリックする）と、所定のログディレクトリに内容が記録され、**直ちにアプリが終了**します。すぐ次の作業に戻れます。
